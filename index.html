Project Scope

Comprehensive overview of our research approach to data-driven learning

Literature Survey

Traditional Learning Management Systems (LMS) apply static or manually defined learning styles that do not adapt in real time to changes in student behavior. Moreover, current dropout monitoring tools are fragmented and reactive; relying on manual intervention and delayed alerts increases operational risk and erodes educator trust.

Research Gap

  • Current monitoring tools are fragmented and reactive.
  • Learning styles rely on static surveys or self-reporting.
  • "Black-box" predictions limit educator trust and actionable intervention.
  • ML models suffer from rapid decay caused by Data and Concept Drift.
  • Infrastructure lacks self-healing optimization for LMS volatility.

Research Problem

Rapidly changing student behaviors and pedagogical shifts cause standard models to lose predictive power quickly. Additionally, the lack of automated systems that transparently infer learning styles and predict disengagement prevents proactive, effective student interventions.

Main Objective

Develop an automated, self-healing MLOps framework and a transparent decision-support system that provides real-time tracking, personalized learning recommendations, and dynamic explainability (XAI) for student engagement.

Sub Objectives

  • Forecast student dropout risks using LMS behavioral data.
  • Decode algorithms via SHAP and LIME for transparent insights.
  • Automatically predict personalized learning styles dynamically.
  • Deploy continuous training via a Self-Healing MLOps pipeline.

Methodology

The methodology encompasses behavioral data extraction, engagement tracking, prediction mapping, model self-healing, and actionable XAI.

01

Engagement Scoring

Algorithms process LMS activity (logins, access duration) to generate measurable engagement metrics for each student.

02

Disengagement Prediction

Supervised machine learning models analyze continuous behavior to identify early signs of dropout risks.

03

Learning Style Extraction

The system automatically infers student learning styles in real-time, bypassing outdated static surveys.

04

Tailored Intervention

Triggers personalized learning paths matching specific preferences to quickly re-engage struggling students.

05

Explainable Analytics (XAI)

A dual-layer XAI engine decodes automated decisions through SHAP and LIME visualizations to highlight root causes.

06

Continuous Training (CT)

A proactive, self-healing MLOps pipeline evaluates data drift and triggers retraining to combat model decay.

07

Interactive Dashboard

Educators review unified insights, actionable alerts, and interpretable visual metrics from a dynamic UI.

Key Pillars

Four Core Platform Modules

A holistic strategy targeting performance insights, data transparency, and robust ML workflows.

Predict

Performance & Dropout Risk

Processes academic and behavioral logs to accurately forecast grades and dropout risks, enabling highly targeted interventions.

Trust

Explainable AI (XAI)

Delivers dynamic interpretability through SHAP and LIME, ensuring that decisions are transparent and actionable for educators.

Adapt

Learning Style Detection

Continuously adapts to diverse learner preferences without static surveys, optimizing real-time content delivery.

Sustain

Self-Healing MLOps

Integrates near-real-time monitoring to mitigate concept drift, preserving effective performance without manual oversight.
